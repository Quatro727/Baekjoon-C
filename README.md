# 백준 문제 C언어 풀이


## 개요
기존에 익힌 C언어를 이용해 **Baekjoon 온라인 저지 사이트**의 단계별 문제 풀이를 통해 c언어에 익숙해지고자 함


## 목적
Baekjoon 단계별 문제를 매일 2문제씩 풀고 풀이를 올려 추후 C언어에 대한 기본 지식이 다시 필요할 때 다시 보고자 함


## 📂 디렉토리 구조 (계속 업데이트 중)

```text
baekjoon-C/
├── StepByStep/      # Baekjoon 단계별 풀이 (자료구조 이전 기초 문제)
├── DataStructure/   # 스택, 큐, 연결 리스트 등 자료구조 구현 및 문제
└── Algorithm/       # 정렬, 탐색, DP 등 주요 알고리즘 문제

------

1. StepByStep 디렉토리에는 날짜별로 디렉토리가 구성
2. DataStructure와 Algorithm 디렉토리는 주제별로 구성될 예정
